# 삼각형 판별하기 

## 필요한 개념 
- 삼각부등식을 알아야한다. <br/>
삼각형의 두 변의 합이 다른 한변의 길이보다 크다는 식이다.<br/>
가능한 세가지 조합 모두 이 정리를 만족한다면 삼각형이 가능하다는 뜻<br/>
1. a + b > c
2. a + c > b
3. b + c > a 

- 삼각형의 세 변의 길이를 a,b,c라고 하면 <br/>
c가 가장 긴 변일 때 a + b > c <br/>
(나머지 두 변의 길이의 합이 가장 긴 변의 길이보다 길다)

2. 나머지 두 수의 합과 같은지를 확인하면 된다.
